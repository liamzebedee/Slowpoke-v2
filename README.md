# Slowpoke
Slowpoke is protocol generator for Golden Gate cloning and colony PCR to be used in Opentrons platforms - OT2 and Flex

This is my fork of the [original repo](https://github.com/Tom-Ellis-Lab/Slowpoke).

I want to see how I can refactor the code to make it as good as possible, based on my 15yrs of software engineering experience.

I'm new to OpenTron so I don't know much about the domain, but I'm very curious to define a powerful typed programming environment for lab automation. I am currently studying MIT's HTGAA.
